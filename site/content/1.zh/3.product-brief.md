## 产品定位

RYMCU BigSmart 是一款面向 AI 语音交互、嵌入式多媒体、物联网控制和掌机娱乐实验的 ESP32-S3 开源硬件平台。开发板以 ESP32-S3-WROOM-1-N16R8 为核心，集成四麦音频输入、扬声器输出、2.4 英寸触摸彩屏、摄像头、六轴姿态传感器、MicroSD 存储、电池供电与外壳结构文件，适合用于小智 AI 终端、离线/联网语音助手、智能家居中控、便携式媒体播放器和掌机原型。

本介绍书面向选型、展示、二次开发和课程教学场景。具体引脚和硬件细节请参阅 [硬件配置说明](/zh/hardware)，上手操作请参阅 [用户详细使用手册](/zh/user-manual)。

## 核心亮点

| 亮点 | 说明 |
|------|------|
| AI 语音交互 | 可运行小智 AI 类语音助手固件，支持按键对话、音频采集、扬声器播放和联网服务 |
| 四麦音频系统 | ES7210 四通道 ADC + ES8311 DAC + NS4150B 功放，支持设备侧 AEC 参考通道 |
| 彩屏触控 | 2.4 英寸 ST7789 LCD，320 x 240 分辨率，配套 GT911 电容触摸 |
| 视觉能力 | GC0308 DVP 摄像头，支持 640 x 480 YUV422 图像采集 |
| 便携多媒体 | MicroSD 卡槽可存放 MP3、视频、图片和游戏资源，支持本地音乐播放与多媒体资源读取 |
| 体感与状态感知 | QMI8658 六轴 IMU 可用于姿态、晃动、交互触发等玩法 |
| 智能家居扩展 | 固件侧可通过 MQTT 工具连接智能家居主题，控制 RGB 灯、加湿器等设备 |
| 开源硬件完整度 | 仓库包含主板、麦克风按键板原理图/EDA 工程，以及 Fusion 360 外壳文件 |

## 硬件组成

| 模块 | 规格 |
|------|------|
| 主控 | ESP32-S3-WROOM-1-N16R8，16 MB Flash + 8 MB PSRAM |
| 音频输入 | ES7210 四通道 ADC，MIC1 主麦克风、MIC2 副麦克风、MIC3 AEC 参考、MIC4 预留 |
| 音频输出 | ES8311 DAC + NS4150B 功放 |
| 显示触摸 | ST7789 2.4 英寸 LCD，320 x 240；GT911 电容触摸 |
| 摄像头 | GC0308，DVP 接口，640 x 480 @ 16 FPS |
| 存储 | MicroSD 卡槽，固件使用 SDMMC 1 线模式 |
| 传感器 | QMI8658 六轴加速度计/陀螺仪 |
| 灯光 | 1 颗 WS2812B RGB LED |
| 按键 | 电源键、Boot/功能键、GPIO10 PTT/自定义键 |
| 供电 | 电池充电管理、充电状态检测、电池电压检测 |

## 软件能力

主要能力包括：

| 能力 | 说明 |
|------|------|
| Wi-Fi 配网 | 首次启动或无有效网络时进入配网；小智工程包含 BluFi 配网说明 |
| 语音对话 | Boot 键单击切换对话状态，GPIO10 支持按住说话 |
| 设备侧 AEC | 固件启用 `CONFIG_USE_DEVICE_AEC` 时，Boot 键双击可切换 AEC |
| 本地音乐 | 支持从 `/sdcard` 播放 MP3，包含播放、停止、列表、上一首、下一首、状态查询等工具 |
| 网络电台 | 支持按电台名称播放网络音频流 |
| 游戏入口 | 支持通过 Launcher 进入固件提供的游戏相关功能 |
| RGB 灯控 | 支持直接设置 RGB 颜色，也可通过 MQTT 灯控主题联动 |
| 智能家居 MQTT | 支持配置 broker、连接、发布、订阅、灯控订阅和加湿器控制示例 |
| IMU 姿态 | 可读取 QMI8658 姿态角、加速度与陀螺仪数据，并支持晃动检测 |
| 摄像头 | 首次请求摄像头时懒加载初始化，降低启动阶段内存压力 |

## 典型应用方案

### AI 语音助手

BigSmart 可作为桌面小智终端。四麦输入和 AEC 参考通道用于更清晰的语音交互，屏幕用于显示状态、表情、菜单或结果，扬声器用于 TTS 播报。

### 智能家居中控

通过 MQTT 工具连接到家庭 broker，语音或 MCP 指令可发布到灯光、加湿器、空调等主题。板载 RGB LED 可作为本地灯控反馈，也可作为 MQTT 灯光示例设备。

### 便携媒体播放器

MicroSD 卡用于存放 MP3 文件。固件提供本地播放、停止、歌曲列表、上一首、下一首和播放状态查询能力，适合做桌面音乐终端或课程实验。

### ESP32-S3 掌机实验

结合 LCD、扬声器、SD 卡和蓝牙 HID 手柄，BigSmart 可用于 ESP32-S3 掌机类实验和自定义游戏演示。

### 多传感互动装置

QMI8658 可读取姿态和晃动事件，摄像头可采集图像，触摸屏和 RGB LED 提供反馈，适合制作互动展示、体感控制或教育实验。
