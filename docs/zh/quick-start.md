# RYMCU BigSmart 快速使用指南

[English](../en/quick-start.md)

本指南用于帮助用户快速完成 BigSmart 的首次开机、联网、服务器选择、固件获取和 SD 卡资源准备。更完整的开发、烧录和外设功能说明请参考 [用户详细使用手册](user-manual.md)。

## 1. 产品外观与接口

BigSmart 板载 2.4 英寸触摸屏、麦克风阵列、扬声器输出、MicroSD 卡槽、USB Type-C 接口、电源键、Boot/功能键和 GPIO10/PTT 自定义按键。

常用部件：

- **电源键**：长按约 3 秒开机或关机。
- **USB Type-C**：用于供电、串口日志和固件下载。
- **MicroSD 卡槽**：用于存放音乐、NES ROM、视频资源等文件。
- **Boot/功能键**：用于配网、对话状态切换、AEC 切换和进入游戏模式。
- **GPIO10/PTT 键**：可作为按住说话按键，具体行为取决于固件。

## 2. 开机

1. 使用 USB Type-C 连接电脑或 5V 电源。
2. 长按电源键约 3 秒。
3. 等待屏幕亮起并进入系统界面。
4. 如需查看日志，可使用串口工具连接对应 COM 口。

## 3. 首次联网

首次使用或清除网络配置后，设备会进入配网流程。常见联网方式包括：

### 3.1 屏幕配网

在设备设置界面中选择 Wi-Fi，输入 2.4G Wi-Fi 名称和密码后连接。

### 3.2 手机连接 AP 配网

设备进入 AP 配网模式后，使用手机连接设备热点，再根据页面提示填写家庭 Wi-Fi 信息。

### 3.3 小程序配网

如果固件支持小程序配网，可按屏幕或串口提示使用对应小程序完成网络配置。

注意：ESP32-S3 仅支持 2.4G Wi-Fi，请不要选择 5G-only 网络。

## 4. 小智 AI 服务器选择

固件可按实际版本支持不同服务器模式，常见选择包括：

| 模式 | 说明 |
|------|------|
| 小智官方 | 使用小智官方服务 |
| RYMCU 官方 | 使用 RYMCU 提供的配套服务 |
| 自定义 | 填写自建或第三方服务地址 |

不同固件版本的入口名称可能略有差异，请以当前设备界面为准。

## 5. 固件获取与下载

可按使用场景选择不同固件：

| 固件来源 | 适用场景 |
|----------|----------|
| 小智官方固件 | 体验小智语音助手能力 |
| RYMCU 官方固件 | 使用 RYMCU 针对 BigSmart 适配的功能 |
| 乐鑫官方示例固件 | 学习 ESP-IDF、外设驱动和基础例程 |

常见下载方式：

1. 使用 ESP-IDF 命令行烧录。
2. 使用 `esptool.py` 烧录合并固件。
3. 使用图形化烧录工具选择串口和固件文件。

详细步骤请参考 [用户详细使用手册](user-manual.md) 的固件编译与烧录章节。

## 6. SD 卡资源更新

建议将 MicroSD 卡格式化为 FAT32，并按功能创建目录：

```text
/sdcard
├── music/      # MP3 音乐
├── nes/        # NES ROM
└── videos/     # 视频转换器生成的 .mjpg/.mp3/.fps 文件
```

视频资源需要先使用 [视频转换器](video-converter.md) 生成 BigSmart 可播放的文件，再复制到 `/videos` 目录。
