# RYMCU BigSmart Quick Start Guide

[中文](../zh/quick-start.md)

This guide helps you complete the first power-on, network setup, AI server selection, firmware download, and SD card resource preparation for BigSmart. For complete flashing, development, and peripheral usage details, see the [User Manual](user-manual.md).

## 1. Product Overview and Interfaces

BigSmart includes a 2.4-inch touch screen, microphone array, speaker output, MicroSD card slot, USB Type-C port, power button, Boot/function button, and GPIO10/PTT custom button.

Common controls and interfaces:

- **Power button**: hold for about 3 seconds to power on or off.
- **USB Type-C**: used for power, serial logs, and firmware flashing.
- **MicroSD card slot**: used for music, NES ROMs, video resources, and other files.
- **Boot/function button**: used for network provisioning, conversation state switching, AEC switching, and entering game mode.
- **GPIO10/PTT button**: can be used as a push-to-talk button, depending on firmware behavior.

## 2. Power On

1. Connect the board to a computer or 5 V power supply through USB Type-C.
2. Hold the power button for about 3 seconds.
3. Wait for the screen to turn on and enter the system UI.
4. To view logs, connect to the corresponding COM port with a serial terminal.

## 3. First Network Setup

On first use, or after network settings are cleared, the device enters a provisioning flow. Common methods include:

### 3.1 On-screen Wi-Fi Setup

Select Wi-Fi in the device settings UI, then enter the 2.4G Wi-Fi SSID and password.

### 3.2 Phone AP Provisioning

When the device enters AP provisioning mode, connect your phone to the device hotspot and follow the page instructions to enter your home Wi-Fi information.

### 3.3 Mini Program Provisioning

If the firmware supports mini program provisioning, follow the on-screen or serial log prompts to complete network setup with the corresponding mini program.

Note: ESP32-S3 supports 2.4G Wi-Fi only. Do not use a 5G-only network.

## 4. Xiaozhi AI Server Selection

Depending on the firmware version, the device may support several server modes:

| Mode | Description |
|------|-------------|
| Xiaozhi official | Use the official Xiaozhi service |
| RYMCU official | Use the RYMCU service adapted for BigSmart |
| Custom | Enter a self-hosted or third-party service URL |

Menu names may vary between firmware versions. Follow the UI shown on your device.

## 5. Firmware Download and Flashing

Choose firmware according to your use case:

| Firmware source | Typical use |
|-----------------|-------------|
| Xiaozhi official firmware | Try Xiaozhi voice assistant features |
| RYMCU official firmware | Use BigSmart features adapted by RYMCU |
| Espressif examples | Learn ESP-IDF, peripheral drivers, and basic examples |

Common flashing methods:

1. Flash with ESP-IDF commands.
2. Flash a merged firmware image with `esptool.py`.
3. Use a GUI flashing tool to select the serial port and firmware file.

See the firmware build and flashing sections in the [User Manual](user-manual.md) for details.

## 6. SD Card Resource Updates

Format the MicroSD card as FAT32 and create directories by feature:

```text
/sdcard
├── music/      # MP3 music
├── nes/        # NES ROM files
└── videos/     # .mjpg/.mp3/.fps files generated by the video converter
```

Video resources must be converted with the [Video Converter](video-converter.md) before being copied to `/videos`.
